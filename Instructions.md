# Lexi - Your AI Education Agent 🧠🌍

Lexi is a powerful, AI-driven educational assistant built for the Hackathon under the **Education and Learning** theme. It uses OpenAI's GPT API to help students learn effectively through interactive and personalized methods.

## Features ✨
1. **Explain Concept:** Get clear, layered explanations for any topic (Beginner, Intermediate, Advanced, or ELI5).
2. **Quiz Me:** Generate custom multiple-choice quizzes on the fly to test your knowledge.
3. **Weak Area Tracker:** Automatically tracks topics you struggle with during quizzes and creates a custom step-by-step study plan.
4. **Assignment Helper:** Paste a complex assignment and Lexi will break it down into actionable, easy-to-follow steps.
5. **Language Practice:** Roleplay in foreign languages (Spanish, French, Japanese, etc.) in various real-world scenarios. Lexi will translate and correct your grammar!

## Tech Stack 🛠️
- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6)
- **Design:** Modern Glassmorphism, Dark Theme, fully responsive.
- **AI Integration:** OpenAI API (`gpt-4o-mini`) via direct REST calls.

## How to Run the App 🚀

Because Lexi is built with pure, lightning-fast Vanilla web technologies, you **do not** need any complicated installations (like `npm install`). 

### Option 1: The Quick Way
Just double-click the `index.html` file in your File Explorer! It will open right up in your default web browser (Chrome, Edge, etc.) and is ready to use immediately.

### Option 2: Using the Windows Batch Script
1. Double-click the `run.bat` file included in this folder.
2. It will automatically start a local server (if you have Python installed) or simply open the `index.html` file for you.

---

## How to Connect the AI (Crucial Step!) 🔑

To make Lexi smart, you need an OpenAI API Key.
1. Get an API key from [OpenAI Platform](https://platform.openai.com/api-keys).
2. Open the Lexi app in your browser.
3. At the top right corner, paste your API key into the input box and click **Connect**.
4. The status indicator on the bottom left will turn green and say "Connected to OpenAI".
5. Your key is safely saved in your browser's local storage so you don't have to enter it every time.
